## Smooth Gallery
This is a gallery module that uses AJAX gallery made by [JonDesign][1] for [LEPTON-CMS][2]

#### Requirements
* [LEPTON-CMS][2], Version >= 4.0.0
* PHP 7.1 (7.2 recommended)
